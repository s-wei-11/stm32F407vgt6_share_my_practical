# 焊板子踩坑记录

## 电烙铁保护

1. 烙铁头不能干烧，会氧化不上锡。
2. 烙铁头有点氧化时：
   - 可以用修复膏；
   - 或弄一个小盒子，里面装点松香和锡，在里面蹭一蹭基本就差不多了（**不要拿刀刮**）。
3. 电烙铁每次用完时，上面要留有锡，防止氧化。

## 常识与焊接技巧

1. 电容、电阻只要是贴片的，都不用分正负，随便贴。  
   （“电解电容”或者带加号的“钽电容”分正反。）
2. 焊接前可在焊盘上挤一点助焊剂，烙铁头贴上去，焊丝往上放即可。  
   （烙铁头不可以在板子上停留太久。）

## 焊接完毕后检查

1. **冷测**：用万用表测通断，尤其是看 5V 和 GND 是否连锡。
2. 通常 PCB 会采用铺铜，这样 GND 所有都是通的。
3. **上电测试**：先上电，测电压稳不稳。  
   正常在 ±0.2V 之间波动都是可以的。
