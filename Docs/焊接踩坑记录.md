# 焊板子踩坑记录
## 电烙铁保护
1. 烙铁头不能干烧 会氧化不上锡
2. 烙铁头有点氧化 可以用修复膏 或 弄一个小盒子里面装点松香和锡 在里面蹭一蹭基本就差不多了（不要拿刀刮）
3. 电烙铁每次用完时上面要留有锡 防止氧化
## 常识的地方
1. 电容电阻只要是贴片的都不用分正负随便贴（“电解电容”或者带加号的“钽电容”分正反）
2. 焊接前可在焊盘上面挤一点助焊剂 然后烙铁头贴上去 焊丝往上放即可（烙铁头不可以在板子上停留太久）

## 焊接完毕后
1. 冷测 用万用表测通断 尤其是看5v 和 gnd是否连锡
2. 通常pcb会采用铺铜 这样gnd所有都是通的 
3. 先上电 测电压稳不稳 正常在0.2之间都是正常的
